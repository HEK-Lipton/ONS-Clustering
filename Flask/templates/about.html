What is this application?

This website is part of my Master Dissertation in Data Science and A.I from the University of Liverpool.

The aim of this project was to produce a web application that allows users to cluster ONS Census data on numerous categories.

What is Clustering?

Clustering is an unsupervised machine learning method. It aims at grouping unlabelled data points together with similar unlabelled data points.

For example on this website when you cluster by Accommodation type, the clustering algorithm firstly gets the data on the accommodation type of each LLA within the UK. This can just be thought of as each LLA having a set of coordinates with n number of dimensions depending on how many attributes there are within a category.

E.g. If the data on the Wirral has , 50 caravans per 1000 households, 700 detached houses per 1000 households, and 200 flats per 1000 households. This could be represented in a three dimensional coordinate space as the Wirral having the coordinates (50,700,200) With these coordinates clustering algorithms can find the distance between different LLA's and determine which should be put in the same cluster. Specific clustering algorithms are discussed on the methods page
